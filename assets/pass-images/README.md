# Pass Images

Place your brand images here. They are used for Apple Wallet passes.

| File | Size | Description |
|------|------|-------------|
| `icon.png` | 29×29 px | App icon (1× — shown on lock screen notification) |
| `icon@2x.png` | 58×58 px | App icon (2×) |
| `icon@3x.png` | 87×87 px | App icon (3×) |
| `logo.png` | 160×50 px | Pass header logo (1×) |
| `logo@2x.png` | 320×100 px | Pass header logo (2×) |

**If files are missing**, the service falls back to a 1×1 pixel placeholder PNG so the pass
can still be generated and tested. Replace with real assets before going live.

All images must be PNG format.
